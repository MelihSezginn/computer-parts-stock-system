# 💻 Bilgisayar Parça Satış ve Stok Takip Sistemi

---

## 📌 Proje Açıklaması

Bu proje, bir bilgisayar satıcısının veya teknik servisinin günlük işlerini (ürün takibi, müşteri kaydı, satış işlemleri vb.) daha kolay ve hızlı yönetebilmesi için geliştirilmiştir.

Amaç, karmaşık görünen veritabanı işlemlerini **basit, anlaşılır ve kullanıcı dostu bir arayüz ile tek bir sistemde toplamaktır.**

---

## 🔐 Giriş Sistemi

Uygulama içerisinde kullanıcıların sisteme giriş yapabilmesi için bir **login ekranı** bulunmaktadır.  
Yetkisiz erişimler engellenerek sistem güvenliği sağlanmaktadır.

---

## 🛠️ Kullanılan Teknolojiler

- C#
- Windows Forms (.NET)
- SQL Server
- ADO.NET

---

## ✨ Özellikler

- 🔐 Kullanıcı giriş sistemi (Login)
- 📦 Ürün ekleme
- 🗑️ Ürün silme
- ✏️ Ürün güncelleme
- 📋 Ürün listeleme
- 👤 Müşteri kayıt sistemi
- 💰 Satış işlemleri
- 🗄️ Veritabanı bağlantısı
